# IBM-HR-Dashboard
----------------------------------------------------------------------------
Employee Performance | HR Metrics | DAX | Data Modeling | Dashboard Design
----------------------------------------------------------------------------
🔹 Overview
This Power BI project focuses on analyzing Human Resources (HR) data to uncover insights about employee performance, experience, and workforce distribution.
The goal was to design an interactive dashboard that helps management track KPIs such as hiring trends, experience levels, and performance rates — all in one place.
